tags:

在 2022 年的时候我写过一一篇博客标题是《为什么我如此期待 Fleet》，Fleet 从 2022 年 10 月 12 号开始 Public Review，一个月后 ChatGPT 正式发布引起全球的 AI 浪潮，ChatGPT 发布即将三年了，而我也删除了电脑里的 IntelliJ IDEA。

大学时候凭借着教育部学籍在线验证报告免费使用了 Jetbrains 的 All Products Pack 五年时间，如果毕业的时候没有 ChatGPT 这样的 AI 工具出现，我想我一定会自己买一套 Jetbrains All Products Pack 而不是转头到各种 AI Editor 和基于命令行的 AI 工具中去，在 2022 年我无比期待的 Fleet 在 2025 年的 11 月依然在 Public Review，上次更新时间停留在 2025 年 5 月 20 日。

其他同时期的 AI Editor 随着大语言模型井喷式的发展：Cursor 和 Claude 强强联手，VS Code 和 Github Copilot 也不甘示弱，又有 OpenAI 大手笔收购 Windsurf；Claude Code 带火 Vibe Code，这两年时间编程工具的进化速度实在是太快了。

这么多 AI Editor 和传统 IDE 之间我选择了 VS Code 作为主力的开发工具来完成除了 iOS 软件开发的工作。从我一开始写代码的时候就一直在使用 VS Code，丰富的扩展生态，极快的软件启动速度，最重要的是强大的 AI 支持和快速的迭代。

现在的我已经不怎么写 Java了，简单的在 VS Code 上使用 Workspace 和 Project Manager 来管理之前的 SpringBoot 项目已经很足够我完成开发了，于是在ChatGPT 发布三年之期，我也删除了电脑里的 IntelliJ IDEA。
