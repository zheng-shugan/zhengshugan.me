## 我使用的硬件
### 工作

主力外设：Nuphy Kick 75、Logitech MX Anywhere 2S

Switch OLED 版是我主力游戏设备，在海拉鲁和胡闹厨房中我度过了快乐 500 小时。最近在 iPad 上购入了一个小丑牌才重新拾起被遗忘的 iPad Pro 来作为游戏设备。



### 生活
我有两部手机，分别是 iPhone 17 和 OPPO Find X9 Pro，我喜欢的软件都在 iOS 上，安卓手机则能体验到目前最强的影像体验。

OPPO Enco X3 和声阔 Q20i 是我的耳机，保证不管在通勤和家中都能获得到一个相对安静的环境。

## 我使用的软件
### 各种 Code
#### Xcode
Xcode 是我开发过程中又爱又恨的软件，想要开发苹果生态内的软件就必须使用到 Xcode 或者来编译运行代码。



过去二十多年的演变，Xcode 已经发展成庞大功能的集合体，从代码编辑、调试、发布到 Xib、Storyboard 众多模块，却又在 AI、插件等功能方面能力的欠缺，希望下一次的 WWDC 就能看到苹果把 Xcode 中的模块进行拆分，降低插件的开发限制，或者再不济加强一下在 VS Code 中的开发体验吧。



#### VS Code
作为一个文本编辑器到 IDE 再到 AI 工具，VS Code 能做的事情太多了，从 JavaScript 到 Ruby，VS Code 在我的编程工作中没有什么是它不能胜任的。

### 写作软件
#### iPhone 备忘录
文字编辑、素材管理、多设备同步这几个看起来简单的功能却一直让我在各种写作软件中拉扯，直到最后还是 iPhone 自带的备忘录笑到最后。



在 iOS26 和 macOS26 能直接将写好的内容直接导出为 markdown 的文件夹，其中还包含了文档中的各种素材，恰好我的博各种有图片、字体都是静态资源，这让我最后的归档工作轻松了许多。

#### 语雀
语雀是我最喜欢的 markdown 编辑器，比起备忘录它在电脑端的编写体验要好太多了，但手机端的体验以及多文档切换我最后还是选择了备忘录作为主要的写作软件，目前只有在上班的时候使用语雀来记录各种内容。

#### Ulysses
Ulysses 是最能让我有沉浸感的工具，还是我赚的不够多，等我赚的的够多了一定买个 Ulysses。

### AI 软件
#### ChatGPT
都 2026 年了我居然还在使用 ChatGPT，文本模型的实力依然是第一梯队又有 macOS 原生开发的客户端，目前用下来最适合我体验的还是 ChatGPT。

#### GLM Code Plan
比起 Claude Code 和 Codex 一个月 20 美元的价格，GLM Code Plan 一个月 20 块钱的价钱显得十分便宜大碗，买的最便宜的 Lite 版本一个月用掉 9 千万的 Token 目前还没遇到被限制使用的情况，也是多亏了 GLM 让我在工作期间基本没怎么手动介入就完成了对屎山代码的重构。
